markdown菜鸟教程

首先来了解一下基本目录和相关文件

  1. 名词说明
    • post：文章
    • page：页面
  2. 目录说明
    • 资源目录（保存post，page的目录）
      • source
      • source/_posts：存放文章的目录
    • 主题目录
      • themes：主题存放目录
      • themes/diaspora：具体主题资源文件存放目录（例如本博客主题：diasppora）
    • 模板目录（保存创建post，page的模板文件，一般不需要更改）
      • scaffolds
  3. 配置文件：
    • 博客配置文件：主目录下的_config.xml
    • 主题配置文件：主题根目录下的_config.xml
